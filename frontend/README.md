# Million Frontend

Next.js 15 application for real estate property management.

## Quick Start

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build
```

## Documentation

See [Frontend Setup Guide](../docs/setup/frontend-setup.md) for detailed instructions.

## Environment Variables

Create `.env.local`:

```env
NEXT_PUBLIC_API_URL=http://localhost:5208
```
