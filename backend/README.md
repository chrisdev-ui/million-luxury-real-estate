# Million Backend API

.NET 9 Web API for real estate property management.

## Quick Start

```bash
# Start MongoDB
docker-compose up -d

# Restore packages
dotnet restore

# Run API
cd src/MillionBackend.API
dotnet run

# Run tests
dotnet test
```
